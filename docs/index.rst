ysphotutilpy
=============

Simple photometry utilities made by ysBach especially for photometry.

.. note::

   This package may undergo backward-incompatible changes. It is a personal
   toolbox used for research and education. See the
   `README <https://github.com/ysBach/ysphotutilpy>`_ for more details.


Installation
------------

Install via pip::

   pip install ysphotutilpy

Or install from source::

   git clone https://github.com/ysBach/ysphotutilpy
   cd ysphotutilpy
   pip install -e .
